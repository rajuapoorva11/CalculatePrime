Program prints the prime factors for a given number on console and generates a output file

USAGE:

1) Python prime_factors.py

a) User will be prompted with  "Enter any Number to find prime factors or type exit to exit :"
b) If user can input a integer or 'exit'
   b1) If user input is a number then corresponding prime numbers are printed
   b2) If 'exit' is give then the program ends.
c) Output of the program is printed in 'result.txt' file
d) The database file is used to store the prime factors of previously searched numbers. Its stored under the name 'Database.txt'

NOTE: Database file contains a dictionary whose keys are previously searched numbers
